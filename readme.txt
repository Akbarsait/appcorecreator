CF AppCore Creator:
-------------------

ColdFusion Builder Extension for Creating Core CF Application files (Application.cfc, error.html, 404.cfm, and index.cfm) 
for ColdFusion MX 7, ColdFusion 8 and ColdFusion 9 versions.
